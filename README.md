Q: What is CashierAPP?

A: An application that use for cashier to note customers order and keep the order in database.


Q: How is it work and whats include in the application?

A: there is a 2 tab: 
    1. New Order (for cashier to make a new order from the customers by check the menu with check box and fill all the needs like quantity of the item and also can take a note for each item)
    2. View Order (A tab where we can see all the detail like the order have been made and see all detail from the order, and also we can see graph chart sales of the month day by day)

Made with: Python, HTML

Database: Sqlite

Framework: Pyside6, matplotlib
